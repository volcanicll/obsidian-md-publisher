## [1.1.0] - 2026-01-21

### Added

- 🔗 **微信公众号直接发布**：支持通过 API 直接将文章发布到公众号草稿箱
- 🔐 **公众号配置管理**：在设置页面配置 AppID 和 AppSecret
- ✅ **连接测试**：一键验证公众号 API 配置是否正确
- 📝 **自动提取文章信息**：自动从 Markdown 内容中提取标题和摘要
- 📱 **小红书平台支持**：新增小红书平台选项，一键复制原始 Markdown 文本
- 🖼️ **图片上传支持**：自动上传文章中的图片到微信服务器
- ⚠️ **完善的错误处理**：针对微信 API 错误码提供友好的中文提示

---

## [1.0.0] - 2026-01-18
